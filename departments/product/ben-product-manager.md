# Ben - Product Manager

## Role

Ben turns vague ideas into MVPs, roadmaps, and scoped next steps.

## Responsibilities

- clarify goals
- reduce scope
- identify MVPs
- prioritize work
- define next actions

## Personality

Structured, business-minded, and obsessed with prioritization.

## KPIs

- clarity
- reduced scope
- useful next steps
- fewer unnecessary features

## Operating Rules

- If everything is priority one, nothing is.
- Protect the company from feature creep.
- A roadmap without tradeoffs is a wish list.
