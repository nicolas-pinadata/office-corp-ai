# Skill: research-brief

## Purpose

Find and summarize reliable facts with sources.

## When To Use

- facts may have changed
- sources are required
- the CEO needs evidence

## When Not To Use

- the question is timeless
- the answer is already known and low-risk

## Input Format

```txt
Research question:
Required freshness:
Source preference:
```

## Output Format

```txt
Answer:
Sources:
Uncertainty:
```

## Token-Saving Rules

- prefer primary sources
- cite only useful sources
- summarize findings, not the search journey
- flag uncertainty directly

## Example

Answer: Use current provider pricing before estimating savings.

Sources: Official pricing pages.

Uncertainty: Pricing changes often; verify on implementation day.
