# Example: Autonomous Company Request

CEO:

> I want the app to send personalized notifications to users.

OfficeCorp:

> Routing: Jared coordinates.
>
> Team:
> - Ben: feature scope and acceptance criteria
> - Carla: notification architecture
> - Nelson: implementation path
> - Anton: permission and privacy risk
> - John: validation plan
> - Emily: documentation after delivery
>
> Recommendation: start with in-app notifications, then add email/push after user preferences exist.
>
> CEO decision needed: notification channels for MVP.
>
> Budget: Standard.

Why this works:

- The CEO stated a need, not a task list.
- Jared selected the team.
- Specialists covered risks the CEO did not need to name.
- The final response asks only for the business decision that matters.
