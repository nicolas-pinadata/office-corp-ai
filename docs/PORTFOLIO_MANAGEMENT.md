# Portfolio Management

Portfolio Management keeps OfficeCorp from chasing too many ideas at once.

The Portfolio Manager tracks ideas, active projects, validated opportunities, rejected opportunities, and strategic priorities.

## Responsibilities

- Maintain the global view of ideas and projects.
- Rank ideas by potential, urgency, complexity, risk, and strategic alignment.
- Prevent the CEO from spreading attention across too many directions.
- Recommend what to prioritize next.
- Archive weak opportunities without losing useful learning.

## Portfolio States

```txt
Captured
-> In Intake
-> In Validation
-> Board Review
-> Approved
-> Pivot Required
-> Research More
-> Rejected
-> In Execution
-> Launched
-> Iterating
-> Archived
```

## Prioritization Inputs

- Market Pain
- Revenue Potential
- Strategic Alignment
- Differentiation
- Execution Complexity
- Acquisition Difficulty
- Risk Level
- Speed to MVP
- CEO urgency
- Existing commitments

## Portfolio Rule

OfficeCorp should recommend fewer active initiatives when focus would improve outcome quality.

Starting a new project should include a portfolio check when it competes with existing priorities.
