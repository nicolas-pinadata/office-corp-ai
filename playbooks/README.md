# Playbooks

Playbooks are optional.

OfficeCorp.ai must work without them. Use playbooks when a project type has recurring patterns, risks, files, or output formats.

## How To Use

1. Pick the playbook that matches the project.
2. Use it after the generic task router.
3. Ignore it if it does not fit the task.
4. Do not create a playbook dependency for simple work.

- n8n
